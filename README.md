# Coding Bootcamp Testimonials Slider

A responsive testimonials slider component built with vanilla HTML, CSS and JavaScript.
Easy to customize and integrate into any static site or single-page app.

## Features

- Automatic synchronization between slides and navigation bullets  
  (Bullets are generated dynamically based on the number of slides.)
- Smooth transition animations
- Responsive design for all screen sizes
- Clean, beginner-friendly HTML, CSS, and JavaScript code
- Easily customizable content and styles
